# Sales & Customer Analytics Dashboard

## Project Overview

Built an end-to-end Sales Analytics solution using Python, MySQL, SQL, and Power BI.

The dashboard provides insights into:

- Executive Overview
- Sales Analysis
- Customer Analytics
- Cohort Analysis
- Churn Analysis
- RFM Segmentation

## Tools Used

- Python
- Pandas
- MySQL
- SQL
- Power BI

## Dataset

Superstore Sales Dataset

- 9,994 Transactions
- 793 Customers
- 2014–2017

## Key Insights

- Technology category generated highest profit
- California generated highest sales
- Consumer segment contributed most revenue
- Customer retention declines after first purchase
